# vite-milkproject

This template should help get you started developing with Vue 3 in Vite.

## 專案說明

### Demo Website : [Link](https://cywcarrie.github.io/vite-milkproject/#/)

模擬電商網站，使用者可以將商品加入購物車、新增或刪除商品和送出訂單，以及可以輸入關鍵字搜尋商品，並且可以瀏覽個別商品的詳細資訊。後台則提供新增及修改商品和優惠券的功能。

## 使用技術

- 使用 Vue.js 框架搭配 Vite 和 Pinia，建立專案
- 引入 Vue Router 和 Vue Axios 等套件，以實現路由管理和 API 請求功能
- 配置 ESLint 符合 Standard 規範，確保程式碼品質
- 透過 Pinia 進行狀態管理，負責跨元件的數據操作與共享
- 利用 Bootstrap 5 和 SCSS 打造響應式界面，提升使用者體驗

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
